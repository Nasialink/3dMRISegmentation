How to run scripts


Create sets:

Change "data_dir" var to point to the correct path
Run "python3 create_sets.py"
"sets" folder should appear

Train zoom in:

Change "resume_model" var to False
Set the config parameters
Run "python3 train.py"
"exp_zoom_in" folder should appear


Train zoom out:

Change "resume_model" var to True
Set the config parameters
Run "python3 train.py"
"exp_zoom_out" folder should appear


Plot metrics:

Run "python3 plt_results.py"
The plots should appear under "exp_zoom_in/figures" and "exp_zoom_out/figures"


Run inference for zoom_in:

Change "resume_model" var to False
The inference images should appear under "exp_zoom_in/inf"


Run inference for zoom_out:

Change "resume_model" var to True
The inference images should appear under "exp_zoom_out/inf"
